Training Tips

Base Model: Start with yolov8n.pt for fast iteration, then try larger models for better accuracy.

Epochs: 100 is a good starting point. Early stopping will prevent overfitting.

Batch Size: Larger batches train faster but need more GPU memory. Start with 16.

Image Size: 640 is standard. Match your inference resolution if possible.
